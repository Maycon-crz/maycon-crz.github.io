RecicladArte - página informativa (em construção)

Site com conteúdos relacionados a ecologia, sustentabilidade, meio Ambiente, reutilização, a idéia é expandir e virar uma plataforma com mais interações entre usuários, onde um certo tipo de usuário vai poder empreender de forma consciente e a favor ao meio ambiente, e o outro tipo ira contribuir mas também será incentivado de uma certa forma.

Tecnologias utilizadas

  • Linguagens: PHP, Javascript, HTML, CSS, SQL;
  • Banco de dados: MySQL;
  • Sistema de Gerenciamento de Banco de Dados(SGBD): Workbench;
  • Frameworks: Bootstrap;
  • Editores usados: Sublimetext e VS CODE;
  • Gerenciador de dependências: Composer;

API

A plataforma conta com uma API criada para integrar o site com o Aplicativo que está sendo criado em Dart/Flutter. No momeno está disponível login, alguns cadastros, edições e exclusões como o de usuário, postagens, charges porém no momento só o administrador pode cadastrar editar e deletar a charge, estão disponíveis também os endpoints para retorno de listagem além dos já citados mais cursos e quiz.

Sobre o site RecicladArte

O site conta com login e cadastro de usuários, recuperação e alteração de senha, cadastro e edição de postagens, exeistem mais opções para o administrador, ao logar para incentivar o acesso existem itens que o usuário recebe, estão sendo planejados uma forma dinâmica para a pessoa ou organização virar membro da alcatéia ecologica, também cadastro de charge logo será liberado para o usuário comum junto com outras formas de interação que tornarão a plataforma uma possivel ferramenta de trabalho para alguns usúaros.
